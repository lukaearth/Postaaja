# Postaaja - Real Estate Listing Automation

Postaaja is an AI-powered platform that helps real estate agents automate their property listing process. Upload images, add property details, and let our fine-tuned GPT-4 AI write perfect Finnish copy in your style.

## Features

- **AI Copy Generation**: Fine-tuned GPT-4 AI creates perfect Finnish property descriptions
- **Multi-Platform Integration**: WhatsApp & Telegram bot support
- **Automatic Posting**: One command publishes everywhere with agent info
- **Style Preservation**: Maintains your unique voice and writing style
- **Time Savings**: Save 90 minutes per post, €430 monthly on average

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone [repository-url]
cd postaaja
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website in your browser.

## Available Scripts

### `npm start`
Runs the app in development mode.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.

### `npm run eject`
**Note: this is a one-way operation. Once you `eject`, you can't go back!**
